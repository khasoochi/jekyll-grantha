# Welcome to Grantha Theme

A beautiful, responsive Jekyll theme designed specifically for Sanskrit, Prakrit, and Pali texts with scholarly apparatus.

## Features

- **Parallel Text Display**: View texts in Devanagari, transliteration, and translation side-by-side
- **Verse Numbering**: Automatic verse numbering with deep-linking
- **Commentary System**: Collapsible commentary sections with support for multiple commentators
- **Reading Progress**: Track your reading progress with a visual indicator
- **Dark Mode**: Easy on the eyes for night reading
- **Search**: Find verses and passages quickly
- **Bookmarks**: Save your favorite passages with localStorage
- **Keyboard Shortcuts**: Navigate efficiently with keyboard commands
- **Mobile Responsive**: Beautiful on all devices
- **Print-Friendly**: Optimized print styles for creating physical copies

## Keyboard Shortcuts

- `t` - Toggle dark/light theme
- `f` - Focus search
- `b` - Toggle bookmark
- `+`/`-` - Increase/decrease font size
- `0` - Reset font size
- `?` - Show this help

## Getting Started

1. Clone the theme repository
2. Install dependencies: `bundle install`
3. Run locally: `bundle exec jekyll serve`
4. Create your content in the `_texts` and `_verses` collections
