# Help & Documentation

## Quick Start Guide

### Reading Texts

1. **Browse** the [Texts](/texts/) page to find available texts
2. **Click** on a text to open the chapter view
3. **Scroll** through verses or use the Table of Contents in the sidebar
4. **Click** verse numbers to get a direct link to that verse

### Using Features

#### Search

- Click the search box in the header or press `f`
- Type to search across all texts, verses, and pages
- Use arrow keys to navigate results
- Press Enter to go to selected result

#### Dark Mode

- Click the theme toggle in the header or press `t`
- Your preference is saved automatically

### Parallel Text Modes

For texts with both Devanagari and Roman scripts, you can toggle between display modes:

1. **Side by Side** - View both scripts simultaneously (desktop)
2. **Stacked** - Scripts stack vertically (mobile)
3. **Toggle** - Click to switch between scripts

Click the layout button to cycle through modes.

### Commentary

Commentaries are collapsible:

- **Click** the commentary header to expand/collapse
- **Multiple** commentators can be shown
- **Different** script options (Devanagari/Roman)

## Navigation

### Breadcrumbs

Follow the breadcrumb trail at the top of pages to understand where you are in the site structure.

### Table of Contents

- **Automatically generated** from page headings
- **Sticky** - stays visible while scrolling
- **Click** any heading to jump to that section
- **Current section** is highlighted

### Chapter Navigation

Use Previous/Next links at the bottom of chapter pages to move between chapters.

## Print

To print a page:

1. Use your browser's print function (Ctrl/Cmd+P)
2. The page will automatically format for printing:
   - Hides navigation and UI elements
   - Expands all commentaries
   - Includes URLs for external links
   - Optimizes typography

## Accessibility

This site is designed to be accessible to all users:

- **Screen Readers**: Proper ARIA labels and semantic HTML
- **Keyboard Navigation**: All features accessible via keyboard
- **High Contrast**: Meets WCAG AA standards
- **Reduced Motion**: Respects prefers-reduced-motion setting
- **Focus Indicators**: Clear focus states for keyboard users

### Screen Reader Tips

- Use heading navigation to jump between sections
- Skip to main content link at the top of each page
- Commentaries have proper ARIA states
- Tables of contents are marked as navigation regions

## Troubleshooting

### Search Not Working

- Make sure JavaScript is enabled
- Check your browser console for errors
- Try refreshing the page
- Clear your browser cache

### Fonts Not Loading

- Check your internet connection (fonts load from Google Fonts)
- Try a different browser
- Clear your browser cache

### Bookmarks Not Saving

- Make sure cookies/localStorage is enabled
- Check if you're in private/incognito mode
- Try a different browser

### Dark Mode Not Saving

- Enable localStorage in your browser
- Check if you're in private/incognito mode
- Clear old data: browser settings â†’ Clear browsing data â†’ Cookies

### Page Not Displaying Correctly

- Try refreshing (Ctrl/Cmd+R)
- Clear cache (Ctrl/Cmd+Shift+R)
- Try a different browser
- Report the issue on GitHub
