# About This Site

This site is built using the **Grantha theme**, a Jekyll theme specifically designed for classical Indic texts including Sanskrit, Prakrit, and Pali literature.

## Features

The Grantha theme provides:

- **Parallel Text Display**: View texts in Devanagari, transliteration, and translation side-by-side
- **Scholarly Apparatus**: Commentary, critical apparatus, and footnotes
- **Modern Reading Experience**: Dark mode, adjustable fonts, bookmarks, and search
- **Accessibility**: WCAG AA compliant, keyboard navigation, screen reader friendly
- **Mobile Responsive**: Beautiful on all devices from phones to large displays

## About the Grantha Theme

Grantha (from Sanskrit ग्रन्थ, "book" or "text") is an open-source Jekyll theme that brings together:

- Traditional scholarship (commentary, critical apparatus)
- Modern web typography and design
- Accessible, responsive layouts
- Progressive enhancement (works without JavaScript)

## Technology

Built with:

- **Jekyll** - Static site generator
- **Vanilla JavaScript** - No dependencies
- **SCSS** - Modular, maintainable styles
- **Progressive Enhancement** - Core functionality works without JS
