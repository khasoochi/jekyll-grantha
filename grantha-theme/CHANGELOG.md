# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-09

### Added
- Initial release of Grantha theme
- Default, text, and verse layouts
- Parallel text display with three modes (side-by-side, stacked, toggle)
- Collapsible commentary system
- Verse numbering with anchor links
- Critical apparatus support
- Footnotes and endnotes
- Dark/light theme toggle with localStorage
- Font size controls (4 levels)
- Reading progress tracker
- Bookmark system with localStorage
- Client-side search functionality
- Keyboard shortcuts for navigation
- Sticky table of contents
- Breadcrumb navigation
- Previous/next chapter navigation
- Ornamental dividers
- Mobile-responsive design (320px to 4K)
- Print-optimized styles
- WCAG AA accessibility compliance
- SEO optimization with jekyll-seo-tag
- RSS feed support
- Comprehensive documentation
- Example content (Bhagavad Gītā Chapter 2)

### Technical Features
- Pure vanilla JavaScript (no jQuery)
- SCSS with CSS custom properties for theming
- Mobile-first responsive design
- Progressive enhancement
- Cross-browser compatibility
- Smooth animations and transitions
- Custom scrollbar styling
- Reduced motion support for accessibility

### Documentation
- Comprehensive README.md
- Installation guide
- Configuration documentation
- Content creation guide
- Customization instructions
- Keyboard shortcuts reference
- Accessibility features
- Browser support information
